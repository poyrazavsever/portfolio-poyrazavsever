# ARC Foreign Trade

**ARC Foreign Trade** is the modern website project I redesigned during my internship for a foreign trade company. It was created to professionally reflect the company's products, services, and corporate identity.

## About

ARC Foreign Trade is a professional web platform operating in international trade and export:

- **Product Catalog**: Detailed product management system  
- **Contact System**: Automated form and notification system  
- **Modern Design**: User‑friendly interface  
- **SEO Optimization**: Search engine–friendly structure

**Blog**
   - Publish blog posts  
   - Edit, delete, archive blog posts  
   - Blog search and statistics  

**Product Management**
   - Category‑based product listing  
   - Detailed product pages  
   - Filtering system  

**Contact System**
   - Contact form  
   - Automatic email notifications  

**Optimization**
   - Fast page loading  
   - Responsive design  
   - SEO‑compatible structure  
   - Image optimization  

## Installation & Usage

```bash
# Clone the project
git clone https://github.com/poyrazavsever/arc_foreign_trade.git

# Enter the project directory
cd arc_foreign_trade

# Install dependencies
npm install

# Start development server
npm run dev
```
